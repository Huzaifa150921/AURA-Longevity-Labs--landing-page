# 🧬 AURA — Clinical Longevity & Biotech Landing Page

AURA is a premium single-page landing page designed for futuristic biotechnology, longevity, regenerative medicine, and luxury healthcare brands.

Built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**, the project combines cinematic animations, glassmorphism, organic morphing backgrounds, and interactive UI components to create a modern, high-end user experience.

---

## ✨ Features

- 🧬 Premium biotech & longevity-inspired UI
- 🌊 Animated organic liquid background blobs
- 🚀 Cinematic splash loading screen
- 🎨 Modern glassmorphism design
- 📱 Fully responsive layout
- 🎯 Interactive diagnostic tabs
- 📈 Animated telemetry canvas
- ✨ Scroll reveal animations
- 🖱️ Mouse-based parallax effects
- 💳 Premium pricing section
- 📝 Modern intake/contact form
- ⚡ Smooth scrolling navigation
- 🎭 Clean typography with Google Fonts
- 🎨 Built entirely with Tailwind CSS

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Google Fonts
- HTML Canvas API
